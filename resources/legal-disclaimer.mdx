## Educational Use Only

IP-Tracker is developed as an educational tool to help security researchers, students, and IT professionals understand:

- How IP geolocation APIs work
- Phone number validation and analysis techniques
- Open Source Intelligence (OSINT) fundamentals
- Network information gathering methodologies

<Info>
  This tool should only be used on systems and data you own or have explicit permission to analyze.
</Info>

## Ethical Guidelines for OSINT

When using IP-Tracker for Open Source Intelligence gathering, follow these ethical principles:

### 1. Legal Compliance

- **Respect Privacy Laws**: Ensure your use complies with local privacy regulations (GDPR, CCPA, etc.)
- **Obtain Consent**: Only analyze phone numbers and IP addresses you own or have permission to investigate
- **No Harassment**: Never use this tool to stalk, harass, or intimidate individuals
- **Professional Use**: Limit usage to legitimate security research, penetration testing (with authorization), or academic studies

### 2. Data Responsibility

- **Secure Storage**: Results are saved locally in `Resultados_Tracker/` - protect this data appropriately
- **No Redistribution**: Don't share personal information obtained through this tool without consent
- **Minimal Collection**: Only gather the information necessary for your legitimate purpose
- **Data Retention**: Delete results when they're no longer needed

### 3. Responsible Disclosure

If you discover security vulnerabilities while using this tool:

1. **Do Not Exploit**: Never exploit vulnerabilities for personal gain
2. **Report Responsibly**: Contact the affected organization through proper channels
3. **Allow Time**: Give organizations reasonable time to patch before public disclosure
4. **Document Properly**: Keep detailed records of your findings and communications

<Warning>
  Unauthorized access to computer systems, networks, or data is illegal in most jurisdictions and can result in criminal prosecution.
</Warning>
